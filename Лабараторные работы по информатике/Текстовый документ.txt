\section{Теоретические сведения:}
\hspace{0.6cm}Удельное сопротивление проволоки круглого сечения, изготовленной из
однородного материала и имеющую всюду одинаковую толщину, расчитывается по формуле:
\begin{equation}
\rho = R_\text{пр}\frac{\pi d^{2}}{4l},
\end{equation}
где $R_{\text{пр}}$ -- сопротивление проволоки, d--диаметр проволоки, l--лина проволоки.
\hspace{0.6cm}Согласно закону Ома для участка цепи:
\begin{equation}
I=\frac{U}{R}
\end{equation}
\vspace{-15pt}
\begin{wrapfigure}{r}{4.7cm}
\includegraphics[width=5cm]{1}
\caption{Схема измерения вольт-амперной характеристики проволоки}
\end{wrapfigure}
\hspace{0.6cm}Для измерения напряжения использовалась схема рис.1. Ввиду неидеальности используемого вольтметра необходимо учесть поправку на его конечное сопротивление $R_V$. Показания амперметра $I_A$ и вольтметра $V_В$ связаны соотношением:

\begin{equation}
V_B = R'I_A,
\end{equation}

где $R'$ -- сопротивление параллельно соединенных проволоки и  вольтметра, причём $\dfrac{1}{R'} = \dfrac{1}{R_\text{пр}}+\dfrac{1}{R_V} \Leftrightarrow R' = \dfrac{R_\text{пр}R_V}{R_\text{пр}+R_V}$, и $R_V \gg R_\text{пр},R'$. График зависимости $V_B(I_A)$ должен представлять прямую, угловой коэффициент которой $R'$, откуда сопротивление образца может быть найдено как:
\newpage
\begin{equation}
R_\text{пр}=\dfrac{R_VR'}{R_V - R'} \thickapprox R'\left( 1+\dfrac{R'}{R_V} \right)    
\end{equation}
\hspace{0.6cm}Формула (4) получена с использованием округления: $\dfrac{1}{1+a} \thickapprox 1-a$.\\
Член, стоящий в скобках в формуле (4), определяет поправку, которую следует внести в измерения.

Таким образом, при измерении диаметра проволоки штангенциркулем случайная погрешность измерений отсутствует. Значит точность результата определяется только точностью систематической погрешности штангенциркуля:
\[\overline{d_1} = (0,4 \pm 0,1)\text{ мм}\]

Измерения с помощбю микрометра содержат как систематическую, так и случайную погрешности:
\begin{itemize}
    \item \textbf{Систематическая}
    \[\sigma_\text{сист} = \Delta_\text{мкм} = \pm 0,01 \text{ мм}\]
    \newpage
    \item \textbf{Случайная}
\begin{enumerate}
    \item \textbf{Среднее значение диаметра:} \[\overline{d_2} =\frac{1}{N} \sum_{j=1}^{N} {d_2}_j = 0,353 \text{ мм}\]
    \item \textbf{Среднее квадратичное отклонение:}
    \[\sigma_\text{отд} = \sqrt{\frac{1}{N-1} \sum_{i=1}^{n}({d_2}_j - \overline{d_2})^2} \thickapprox 0,00955 \text{ мм}\]
    \item \textbf{Средняя квадратичная погрешность отдельного измерения }
    \[\sigma_\text{сл} = \frac{\sigma_\text{отд}}{\sqrt{N}} \thickapprox 0,00302 \text{ мм}\]
    \item \textbf{Суммарная погрешность измерения диаметра проволоки}
    \[\sigma_\text{полн} = \sqrt{\sigma_\text{сист}^2 + \sigma_\text{сл}^2} \thickapprox 0,0104\text{ мм}\]
\end{enumerate}
\end{itemize}

Таким образом, поскольку $\sigma_\text{сл}^2 \ll \sigma_\text{сист}^2$, то можно считать проволоку однородной по диаметру, а погрешность при измерении диаметра проволоки микрометром определяется только точностью систематической погрешности микрометра.

Окончательные результаты измерения диаметра проволоки:

\hspace{1.5cm}\underline{Штангенциркулем:} $d_1 = 0,4 \pm 0,01 \text{ мм.}$

\hspace{1.5cm}\underline{Микрометром:} $d_2 = 0,353 \pm 0,01 \text{ мм}$ ($\varepsilon_{d_2} = 2,83 \%$).

\subsection{Определения площади поперечного сечения проволоки:}
\hspace{0.6cm}Для этого воспользуемся значением диаметра проволоки, полученным микрометром, так как оно более точно, и соответствующей формулой:
\[S = \frac{\pi d_2^2}{4} \thickapprox 0,978\cdot 10^{-3}\text{ см}^2\]

Велечину погрешности площади поперечного сечения найдем по формуле:
\[\sigma_S = 2\frac{\sigma_\text{полн}}{d_2}S \thickapprox 5,3\cdot 10^{-5} \text{ см}^2\]

Для каждой длины l строим апроксимирующие прямые, определяя их угловой коэффициент $\overline{R}$, используя метод наименьших квадратов, по формуле:
\[\overline{R} = \frac{\langle VI\rangle}{\langle I^2\rangle}\]
где угловые скобки означают усреднение. Для каждой велечины значения для соответствующего размера указаны в табл. 5.

Среднеквадратичную случайную погрешность определения углового коэффициента вычисляем как:
\[\sigma_{\overline{R}}^{\text{сл}} = \sqrt{\frac{1}{N-1}\left(\frac{\langle V^2\rangle}{\langle I^2\rangle} - \overline{R}^2\right)}\]

Оценим возможную систематическую погрешность, обусловленную инструментальными погрешностями приборов. Предполагая, что при всех измерениях относительная погрешность одинакова, оценим погрешность вычисления частного R = V/I при максимальных значениях V и I:
\[\sigma_{\overline{R}}^{\text{сист}} = R\sqrt{\left(\frac{\sigma_V}{V_{max}}\right)^2 + \left(\frac{\sigma_I}{I_{max}}\right)^2}\]
где $\sigma_V$ и $\sigma_I$ -- средние квадратичные ошибки измерения вольтметром и амперметром. Результаты заносим в табл. 6.

Для вольтметра средняя квадратичная ошибка вычесляется как половина абсолютной погрешности вольтметра:
\[\sigma_V = \frac{\Delta_B}{2} = 0,6мВ\]
А для амперметра аналогично:
\[\sigma_I = \frac{\Delta_A^\text{ср}}{2} = 0,1 \text{ мA}\]
где $\Delta_A^\text{ср}$ -- средняя абсолютная погрешность амперметра($\Delta_A^\text{ср} = \frac{\Delta_{A_{\text{min}}} + \Delta_{A_{\text{max}}}}{2} = 0,2$ мА).

Полная погрешность измерения $\overline{R}$ равна:
\[\sigma_{\overline{R}}^{\text{полн}} = \sqrt{(\sigma_{\overline{R}}^{\text{сист}})^2 + (\sigma_{\overline{R}}^{\text{сл}})^2}\]

Для всех трёх длин l вносим поправку в измеренное значение сопротивления по формуле:
\[R_\text{пр} = \overline{R} + \frac{\overline{R}^2}{R_B}\]
где $R_B$ -- внутреннее сопротивление вольтметра. Ввиду малости поправки считаем $\sigma_{\overline{R}}^{\text{пр}} = \sigma_{\overline{R}}^{\text{полн}}$.
